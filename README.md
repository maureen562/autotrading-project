# Autotrading Project

This repository contains an automated trading project using the Korea Investment Open API.

## Files

- `autotrader_thursday.py`: first version executed on Thursday
- `TRADING_LOG_THURSDAY.md`: trading log generated by the Thursday version
- `autotrader_friday.py`: stabilized version executed on Friday
- `TRADING_LOG_FRIDAY.md`: trading log generated by the Friday version

API keys, API secrets, and account numbers are replaced with placeholders.
